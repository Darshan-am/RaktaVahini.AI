# RaktaVahini.AI - Healthcare Platform

RaktaVahini.AI is an AI-powered healthcare and blood donation platform.

## Features
- Login & Registration
- Blood Donor Search
- Blood Request System
- Healthcare Assistance
- Emergency Notifications

## Technologies
- Kotlin
- Jetpack Compose
- Firebase
- Android Studio

## Installation

git clone https://github.com/Darshan-am/RaktaVahini.AI.git

Open in Android Studio and run the project.
